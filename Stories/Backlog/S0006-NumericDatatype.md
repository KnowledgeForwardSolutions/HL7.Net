# S0006-NumericDatatype

As a developer who uses HL7.Net, I want the HL7 NM datatype to be fully supported
when parsing messages.

## Requirements

- DecimalField class and IntegerField class that implement NM datatype as per HL7 V2.2 spec section 2.8.10.4

## Definition of DONE

1. DecimalField class
2. Unit tests for DecimalField method
3. IntegerField class
4. Unit tests for IntegerField method
